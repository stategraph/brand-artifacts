# Stategraph Brand Artifacts

Official Stategraph brand assets, logos, and design resources.

## Logo Files

### Wordmark (Icon + Text)
`logos/wordmark/` - Full logo with icon and "STATEGRAPH" text

- **Blue:** `stategraph_logo_wordmark_blue.svg` (Primary)
- **Black:** `stategraph_logo_wordmark_black.svg` (Light backgrounds)
- **White:** `stategraph_logo_wordmark_white.svg` (Dark backgrounds)

Each available in SVG, PNG, and JPG formats.

**Usage:** Primary brand representation in headers, marketing materials, presentations.

### Symbol (Icon Only)
`logos/symbol/` - Graph icon without text

- **Blue:** `stategraph_logo_symbol_blue.svg` (Primary)
- **Black:** `stategraph_logo_symbol_black.svg` (Light backgrounds)
- **White:** `stategraph_logo_symbol_white.svg` (Dark backgrounds)

Each available in SVG, PNG, and JPG formats.

**Usage:** Favicons, app icons, social media avatars, tight spaces where text doesn't fit.

### Favicons
`logos/favicon/` - Web favicon assets

- `favicon.svg` - SVG favicon (modern browsers)
- `favicon.png` - 64x64 PNG favicon
- `apple-touch-icon.png` - 180x180 iOS home screen icon

**Usage:** Website favicons and mobile web app icons.

## Logo Guidelines

### DO
- ✅ Use the blue logo as the primary brand mark
- ✅ Maintain aspect ratio when scaling
- ✅ Use white logo on dark backgrounds
- ✅ Use black logo on light backgrounds
- ✅ Give the logo breathing room (minimum clear space)
- ✅ Use symbol for favicons and small spaces
- ✅ Use wordmark for primary brand placement

### DON'T
- ❌ Distort or stretch the logo
- ❌ Change the logo colors (except approved variants)
- ❌ Add effects (drop shadows, gradients, outlines)
- ❌ Rotate the logo
- ❌ Place logo on busy backgrounds
- ❌ Recreate or modify the logo

## File Format Guide

- **SVG:** Scalable vector format - use for web and when scaling is needed
- **PNG:** Raster format with transparency - use for presentations and documents
- **JPG:** Raster format without transparency - use when file size matters
